# Flutter Age Calculator Reference Project

This directory contains a complete, production-ready, fully functional **Flutter Age Calculator** project with a matching Material 3 design and offline calculator logic.

## Flutter Features
- **Material 3 UI**: Matches the exact color schemes, elevation layers, and sleek aesthetic of the native Kotlin Jetpack Compose app.
- **Theme toggle**: Complete support for Light and Dark modes.
- **Detailed Calculations**: Returns exact age in years, months, and days, along with a circular next birthday countdown timer, day of birth, and comprehensive live metric tracking.
- **AdMob Ready**: Preconfigured bottom layout ready for Google Mobile Ads integration.

## How to Run This Flutter Project

1. Ensure Flutter is installed on your local development machine: [Flutter Installation Guide](https://docs.flutter.dev/get-started/install).
2. Create a new Flutter project or integrate this code into your existing project.
3. Replace your project's `lib/main.dart` with the contents of `flutter_reference/lib/main.dart`.
4. Install `intl` formatting package inside your Flutter directory:
   ```bash
   flutter pub add intl
   ```
5. Run the application:
   ```bash
   flutter run
   ```

Enjoy calculating ages beautifully across both Android Native and Flutter!
